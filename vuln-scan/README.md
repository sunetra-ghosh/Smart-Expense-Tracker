# Continuous Vulnerability Scanning & Patch Management

This tool automates vulnerability scanning and patch deployment for financial APIs, with dashboards for remediation tracking.

## Modules
- vulnerability-scanner.js: Scans APIs for vulnerabilities
- patch-deployment.js: Automates patching
- remediation-dashboard.js: Tracks remediation status
- api-integration.js: Integrates scanning and reporting
- vuln-scan-ui.js: CLI demo

## Usage
Run vuln-scan-ui.js to see a demo of the scanning and patching workflow.
