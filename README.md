
# 💸 Smart Expense Tracker

A simple full-stack web application for tracking personal income and expenses. The system helps users manage financial data, monitor transactions, and gain basic insights into spending patterns.

---

## 🚀 Features

* Add, update, and delete income/expense records
* View balance and transaction history
* Basic analytics and insights
* Responsive user interface
* Secure data handling

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Database: MongoDB

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/expenseflow.git
cd expenseflow
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm start
```
